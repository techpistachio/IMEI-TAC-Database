# Email Templates

## Approval email

Subject:

`Your TACDatabase API key is approved`

Body should include:

- greeting
- approval note
- API key
- base URL
- auth header format
- request example
- rate limits
- support email

## Rejection email

Subject:

`Your TACDatabase API request was not approved`

Body should include:

- polite decline
- short reason
- support contact
- re-apply option, if allowed
